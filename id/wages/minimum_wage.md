## Summary
Indonesia has **no single national minimum wage**. It is set annually, effective **1 January**, as a **provincial minimum wage (UMP — Upah Minimum Provinsi)** by each Governor, with higher **regency/city minimum wages (UMK)** where set. For **2026**, the increase moved from a flat national percentage to a **per-region growth-based formula** (PP 49/2025), so each province's rate differs. The **Jakarta (DKI) UMP 2026 = Rp5,729,876/month** (the national anchor).

> Currency: **Rp / IDR**. Set by Governors under **PP 36/2021 as amended by PP 49/2025**.

> `volatility: continuously-changing` — UMP/UMK are reset every year (decreed ~Nov–Dec, effective 1 Jan). **Re-pull the current year's Governor decree (SK Gubernur) before relying on a figure.**

## How it works
| Tier | What |
|------|------|
| **UMP** (Upah Minimum Provinsi) | Provincial floor, set by the Governor (SK Gubernur), effective 1 Jan. 38 provinces. |
| **UMK** (Upah Minimum Kabupaten/Kota) | Regency/city floor, set by the Governor, must be **higher than the UMP**. Where none is set, the UMP applies. |
| **UMSP** | Sectoral minimum wage (revived under PP 49/2025), set above the UMP for specified sectors. |
